# D2 Diagrams

## Basic connection

```d2
x -> y
```

## Sequence diagram

```d2
shape: sequence_diagram

alice -> bob: What does it mean?
bob -> alice: No idea
bob -> charlie: Do you know?
charlie -> bob: Yes!
```

## Container nesting

```d2
server: {
  app: Application
  db: Database
}

client: Browser

client -> server.app: HTTP request
server.app -> server.db: SQL query
```

## Class diagram

```d2
direction: right

Animal: {
  shape: class
  name: string
  sound(): string
}

Dog: {
  shape: class
  breed: string
  fetch(): void
}

Cat: {
  shape: class
  indoor: bool
  purr(): void
}

Animal <-- Dog
Animal <-- Cat
```

## Grid layout

```d2
grid-rows: 2

a: Build
b: Test
c: Deploy
d: Monitor
```
